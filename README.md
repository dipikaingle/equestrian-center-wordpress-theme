## 📌 Overview
Equestrian Center is a custom WordPress theme designed for horse riding schools, equestrian clubs, and training centers.

The theme provides a clean, engaging, and professional layout to showcase services, facilities, and events.

Ideal for:
- Equestrian centers
- Horse riding schools
- Training academies
- Sports and activity clubs

---

## ✨ Features
- Fully custom WordPress theme (no page builder dependency)
- Responsive design across all devices
- Service and facility showcase sections
- Event and activity display
- Clean and user-friendly UI/UX
- SEO-friendly structure
- Optimized performance and fast loading
- Cross-browser compatibility

## ⚙️ Installation
1. Download or clone the equestrian-center-pro
2. Upload the theme folder to:
   `/wp-content/themes/`
3. Go to WordPress Admin → Appearance → Themes
4. Activate the theme
